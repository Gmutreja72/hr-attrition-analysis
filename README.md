# HR Attrition & Analytics Dashboard
**Tools:** IBM Cognos Analytics · SPSS · Excel
**Date:** January 2023

## Overview
Analyzed 1,400+ IBM employee records to identify the key drivers of employee attrition.
Built an interactive IBM Cognos dashboard to help HR teams monitor turnover trends in real time.

## Key Findings
- Top 5 attrition drivers: overtime, low job satisfaction, short tenure, distance from home, low income
- Departments with highest attrition: Sales (24%), HR (19%)
- Recommended 3 policy changes projected to reduce attrition by 18%

## Dashboard Features
- Attrition rate by department, job role, and tenure group
- Overtime vs. attrition correlation chart
- Quarterly trend tracking
